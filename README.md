# 谵妄亚型四分类机器学习模型

## 项目简介

这是一个用于谵妄亚型四分类的机器学习模型训练项目。该模型能够识别和分类四种不同的谵妄亚型，为临床诊断提供辅助决策支持。

## 功能特点

- **四分类预测**：准确识别四种谵妄亚型
- **机器学习模型**：基于先进的机器学习算法训练
- **可解释性分析**：提供模型决策的可解释性分析
- **临床应用**：为医疗专业人员提供诊断辅助
